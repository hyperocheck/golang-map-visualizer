Это консольная программа с веб-визуализацией, которая наглядно показывает, как работает тип данных *map* в Go версии `<=1.23.x`. Оригинальный код работы map можно найти в исходниках языка [здесь](https://cs.opensource.google/go/go/+/release-branch.go1.23:src/runtime/map.go). Код тестировался и разрабатывался на версии `1.23.12`, поэтому настоятельно рекомендуется использовать именно эту версию.

## Как скачать, установить и запустить
```shell
go install golang.org/dl/go1.23.12@latest
go1.23.12 download
git clone https://github.com/hyperocheck/golang-map-visualizer.git
cd golang-map-visualizer
go1.23.12 run ./cmd/visualizer/
```
## Как использовать
Входная точка программы находится в файле `./cmd/visualizer/main.go`. Здесь создаётся *map* типа `map[string]int`, который можно предварительно заполнить данными. Вы можете использовать любой сериализуемый тип в качестве ключа и значения. Из коробки полный функционал консоли поддерживают типы `string`, `bool` и `int` (со всеми его вариациями) в любых комбинациях ключ значение. Если хотите использовать кастомный тип данных и при этом так же использовать весь функционал консоли, то необходимо реализовать для вашего типа метод `FromIndex` интерфейса `Generatable` и метод `Parse` интерфейса `Parseable` (пример можно найти в `./cmd/visualizer/example.go`)
```go
func main() {    
    m := make(engine.Map[string, int])

    /*
    // Пример использования флагов:
    // go1.23.12 run ./cmd/visualizer --from 100 --to 1000 
        for i := cmd.Flag.From; i < cmd.Flag.To; i++ {
            m[intToStr(i)] = i
        }
    */

    work(m) // не удалять :)
}
```
## Как играть
Перейдите в браузер на `http://localhost:8080` и откройте CLI. В консоли работает история (стрелки ↑/↓) и автодополнение через Tab. Далее покажу примеры команд и что они делают.  
```shell
# Вставить пару ключ знчение
# То же самое, что map[123] = 382783728
insert 123 382783728
```
```shell
# Обновить пару ключ знчение
# То же самое, что map[123] = 9990
update 123 9990
```
```shell
# Удалить пару ключ знчение
# То же самое, что delete(map, 123)
delete 123
```
```shell
# Вывести все пары ключ значение
show
```
```shell
# Вывести состояние структуры hmap
hmap
```
```shell
# Показать доступные команды
help
```
```shell
# Завершить работу
# Или с помощью ctrl+c
exit
```
```shell
# Вставить пары ключ значение в цикле от 10 до 1400
# То же самое, что for i := 10; i++; i <= 1400 {map[i] = i}
range insert 10 1400
```
```shell
# Вставить пары ключ значение в цикле от 10 до 1400 в лайф режиме визуализации
# То же самое, что for i := 10; i++; i <= 1400 {map[i] = i}
range insert 10 1400 --life
```
```shell
# Удалить пары ключ значение в цикле от 10 до 1400
# То же самое, что for i := 10; i++; i <= 1400 {delete(map, i)}
range delete 10 1400
```
```shell
# Режим коллизий
# Вставить 10 каких-то пар ключ значений специально в первый бакет (bid=0)
evil 10
```
```shell
# Режим коллизий
# Вставить 10 каких-то пар ключ значений специально в первый бакет (bid=0) в лайф режиме
evil 10 --life
```
```shell
# Режим коллизий
# Вставить 10 каких-то пар ключ значений специально в 3 бакет (bid=3) в лайф режиме
evil 10 --bid 3 --life
```
```shell
# Вывести подробный step-by-step процесс поиска значения под ключом 2382
# Соотвествует реальной работе оригинальной функции mapaccess1
mapaccess1 2382
```
